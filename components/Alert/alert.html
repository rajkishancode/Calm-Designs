Alerts

Alerts are used to communicate a state that affects a system, feature or page.

Alert Variants : | Success| Info | Warning | Error |

Success! This alert box could indicate a successful action.

Info! This alert box could indicate a neutral action.

Error! This alert box could indicate a error or negative action.

Warning! This alert box indicate warning that need attention.
